# coding_challenge

## Installation

npm packages need to be installed first.

```sh
> cd frontend
> npm install
```

```sh
> cd ..\backend\
> npm install
```

## Start services

Please use npm scripts to start services.

Example:

```sh
cd frontend
npm run dev
```

or

```sh
cd backend
npm run start:dev
```

## Testing

Please also use npm scripts for running tests
You need to change to frontend or backend folder.

```
npm run test:watch
```
